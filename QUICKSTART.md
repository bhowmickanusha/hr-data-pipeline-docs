# Quick Start Guide

## Prerequisites
- Azure subscription with ADF
- Databricks workspace with UC
- Oracle DB with SHIR configured

## Steps
1. Clone both repos
2. Deploy ADF pipelines
3. Deploy Databricks notebooks
4. Configure Power BI connection
5. Trigger first run

## Verify Success
- Check ADLS for parquet files
- Query UC Delta Tables in Databricks
- Check Power BI dashboard
